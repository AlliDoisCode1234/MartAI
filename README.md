# MartAI - AI-Powered SEO Automation Platform

MartAI automates your SEO content system with AI-powered keyword clustering, quarterly planning, content briefs, and automated publishing to WordPress, Shopify, and Webflow.

## Quick Start

```bash
npm install
cp .env.example .env.local  # Add your keys
npx convex dev              # Initialize backend
npm run dev                 # Start dev server
```

## Features

- Customer intake with GA4/GSC connection
- AI-powered keyword clustering
- 12-week quarterly planning with drag-drop
- AI-generated content briefs and drafts
- Automated publishing to WordPress/Shopify/Webflow
- Analytics dashboard with insights
